### 台阶的走法

> 有n个台阶，一次可以迈一个台阶；也可以迈两个台阶；问有多少走法

```go
func f(n int)int {
    if(n==1){
        return 1
    }
    if(n==2){
        return 2
    }
    return f(n-1)+f(n-2)
}
```

> 上述算法有重复计算了；可以用两个变量来动态存储；比如f(3)=f(2)+f(1)
